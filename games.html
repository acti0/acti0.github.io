Recursive Rescale

Games | 24/08/2024

I've released my first game! It was made during the GMTK24 Game Jam (Theme: Built to Scale).

It's a little first person puzzle platformer game where the player is in an endlessly repeating world. The smaller world affects the bigger one, so the player can build their path up a mountain.
